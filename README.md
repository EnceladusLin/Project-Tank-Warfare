## 📖 项目简介

**《装甲前线：视觉重制版》** 是一款融合了**赛博朋克美学**与**MOBA技能机制**的俯视角战术坦克射击游戏。在霓虹闪烁的全息战场上，驾驶未来战车，击败敌方势力，守卫联盟基地。

### 🎮 核心特色

- **🌐 动态天气系统**：四种天气（晴天、离子风暴、暴雨、冰冻）每120秒切换一次，直接改变战场地形与战术
- **⚔️ MOBA式技能体系**：冲刺闪避（Shift）、相位跳跃（Q）、能量护盾（Space）、EMP脉冲（F）、黄金狂暴（C）、战术核弹（B）
- **🎨 赛博朋克视觉**：全息雷达、动态扫描线、故障艺术效果、霓虹光效
- **🤖 智能AI对手**：五种特殊职业坦克（隐形烟雾、地雷轰炸、冰冻控制、全屏闪光）+ 三阶段Boss战
- **🗺️ 战略地图**：4阵营基地对抗、中央Hub争夺点、Buff守卫、地形互动（水域可冻结、森林加速、墙体可摧毁）

## 🎯 游戏玩法

### 🏆 胜利条件

摧毁所有敌方基地，同时保护己方**联盟基地（蓝色）**存活。

### ⏱️ 时间限制

每局游戏8分钟（480秒），超时则以**剩余基地血量**判定胜负。

## 🎮 操作指南

### 移动与战斗

| 按键 | 功能 |
|------|------|
| **W/A/S/D** | 移动坦克 |
| **鼠标移动** | 瞄准炮塔 |
| **左键** | 主武器射击 |
| **右键** | 发射导弹（最多3发，5秒充能1发） |
| **滚轮上/下** | 切换武器（主炮/激光/机枪） |

### 技能系统

| 按键 | 技能名称 | 冷却时间 | 效果 |
|------|----------|----------|------|
| **Shift** | 引擎过载（TURBO） | 3秒 | 移动速度+15%，持续10秒 |
| **Q** | 相位跳跃（DASH） | 10秒 | 向前方瞬移一段距离 |
| **Space** | 能量护盾（SHIELD） | 8秒 | 生成2秒护盾，免疫所有伤害 |
| **F** | EMP脉冲（EMP） | 15秒 | 净化所有负面状态（冰冻/燃烧/致盲） |
| **C** | 黄金狂暴（RAGE） | 180秒 | 伤害+30%、速度+30%，持续40秒 |
| **B** | 战术核弹（NUKE） | - | 解锁后可用，范围AOE爆炸 |

### 系统功能

| 按键 | 功能 |
|------|------|
| **P** | 暂停游戏 |
| **L** | 阵亡后重生（消耗击杀数） |
| **R** | 游戏结束后重启 |

## 🌦️ 天气系统详解

游戏每**120秒**自动切换天气，天气直接影响战场机制。

| 天气类型 | 视觉效果 | 战术影响 |
|----------|----------|----------|
| **☀️ 晴天（CLEAR）** | 无滤镜 | 无特殊效果，标准战斗 |
| **⚡ 离子风暴（ION_STORM）** | 紫色电磁脉冲 | 雷达信号干扰，敌我位置闪烁 |
| **🌧️ 暴雨（RAINSTORM）** | 深蓝滤镜+雨滴 | 视野降低，水域流动加速 |
| **❄️ 冰冻（FREEZE）** | 冰蓝渐变覆盖 | **所有水域冻结成可通行冰面**，移动速度-50% |

### ❄️ 冰冻天气核心机制

- **地形改变**：原本不可通行的水域全部变为冰面，开辟新的进攻/撤退路线
- **移动惩罚**：在冰面上移动速度降低50%
- **战术应用**：
  - ✅ 利用冰桥突袭敌方水域后方基地
  - ✅ 引诱敌人上冰面，己方在陆地作战获得速度优势
  - ❌ 避免在冰面正面硬刚

## 🗺️ 地图要素

### 基地系统

地图上有**4个阵营基地**：

| 阵营 | 位置 | 颜色 | 血量 |
|------|------|------|------|
| 🔵 **蓝色联盟** | 底部中央 | 青色 | 3750 |
| 🔴 **红色军团** | 顶部中央 | 红色 | 3750 |
| 🟢 **绿色先锋** | 左侧中央 | 绿色 | 3750 |
| 🟣 **紫色虚空** | 右侧中央 | 紫色 | 3750 |

- **基地防御**：自动向敌方单位发射双联炮
- **胜利条件**：摧毁除蓝色外的所有敌方基地

### 中央Hub争夺点

- **位置**：地图中心（60, 40）
- **功能**：
  - 占领后每10秒自动修复150血量
  - 武器冷却时间-25%
  - 提供战术雷达增强
- **机制**：站在Hub半径内（150像素）逐渐占领，离开后逐渐流失控制权

### Buff守卫单位

在游戏**60秒后**，地图四角刷新**Buff守卫**：

| 守卫类型 | 颜色 | 击杀奖励 |
|----------|------|----------|
| 🔴 **红色守卫** | 红色 | **燃烧Buff**：武器附带火焰伤害，免疫冰冻 |
| 🔵 **蓝色守卫** | 青色 | **速度Buff**：移动速度+50% |

- **血量**：3750
- **重生时间**：击杀后120秒重生
- **持续时间**：永久（直至阵亡）

### Boss单位

游戏**200秒后**，地图五个竞技场刷新Boss：

| Boss类型 | 血量 | 特性 |
|----------|------|------|
| **普通Boss** | 1875 | 高伤害，中等速度 |
| **Overlord（霸主）** | 10000 | 三阶段变化，弹幕攻击，传送能力 |

- **奖励**：大量经验值，解锁战术核弹（B键）
- **战术**：推荐组队围杀，单人挑战难度极高
